Name
	sctp get singlethread 

SYNOPSIS
	sctp get singlethread stackname <stack-name>

DESCRIPTION
	Returns true if stack is single threaded else false
	
PARAMETERS

	Optional Parameters

	<stack-name>	-	Name of the stack on which this command is executed. 
					If not passed, the first stack configured in ShellExecutor
					will be used.		

EXAMPLES
	sctp get singlethread
